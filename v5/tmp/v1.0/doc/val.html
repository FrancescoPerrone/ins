val.pl -- Values and evaluation function

This file provides defintion for Hal's set of values and also provides an evalution function which defines the status of a value abscribed by Hal to the transition between two states.

better(-A:state, -B:state, ?Val) is det
Defines the value's status 'better'

It means that in state state 'B' the status of 'Val' is 'better' under Hal's point of view.

Arguments:
A- a state
B- a state
Val- a value
promote(?Ini, ?Fin, ?V)
Promotion function Defines what it takes for a value to be promoted in a transition.
Arguments:
Ini- the initial state
Fin- the new state
V- indicates that V is promoted
demote(?Ini:state, ?Fin:state, ?V:value)
Demotion function Defines what it takes for a value to be demoted in a transition.
Arguments:
Ini- the initial state
Fin- the new state
V- indicates that V is promoted
eval(?Ini:state, ?Fin:state, -L:list)
Evaluation set. Create a set containing an evaluation of what has been promoted/demoted in a transition, under Hal's point of view.
Arguments:
Ini- an initial state
Fin- a new state
L- a list of value's status
