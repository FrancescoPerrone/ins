trans.pl -- Transition system

This file defines a set of rules for Hal-Carla scenario to perform transition from state to state.

trans(?Ini:state, ?Ac, ?New:state, +N:int)
Transit of N step/s, from p to reach q. Means that from state 'Ini', we apply 'Ac' to reach state 'New' which is 'N' step/s head of 'Ini'.
Arguments:
Ini- initial state
Ac- , performing action
New- a new state of affairs
N- int specifying the number of step/s taken by the transition
See also
- action.pl
