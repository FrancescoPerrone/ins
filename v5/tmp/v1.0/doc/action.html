action.pl -- Action precondition and partial transition function

This file provides definitions for actions, action's precondition and partial transition function.

author
- Francesco Perrone
See also
- precond/2 the precondition function
- perform/3 the partial transition function

e.g.

?- precon(buy, State).
State = [0, 1, 1, _G2090, _G2093, _G2096].

?- perfor(Ini, Fin, buy).
Ini = [0, 1, 1, _G2104, _G2107, _G2110],
Fin = [1, 0, 1, _G2104, _G2107, _G2110].
true.

precond(?Action, ?State:state) is semidet
Action's precondition function. Meaning that 'Action' can be performed from 'State'.
Arguments:
Action- an action's name (ground)
State- a action's precondition state
perform(?Ini:state, ?New:state, ?Action) is semidet
Partial transition function. 'New' is the state tha would result by performing 'Action' from 'Ini'.
Arguments:
Ini- an initial state.
New- a valid new state.
Action- action name (ground).
