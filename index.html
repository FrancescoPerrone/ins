Notes for me

An Action-based Alternating Transition System (AATS) is an (n + 7)-tuple:

  S = <hQ, q0, Ag, Ac1, ..., Acn, ρ, τ, φ, πi>    where:
  • Q is a finite, non-empty set of states;
  • q0 ∈ Q is the initial state;
  • Ag = {1, ..., n} is a finite, non-empty set of agents;
  • Aci is a finite, non-empty set of actions, for each i ∈ Ag where Aci∩Acj = ∅ for all i 6= j ∈ Ag;
  • ρ : AcAg → 2Q is an action precondition function, which for each action α ∈ AcAg defines the set of states ρ(α) from which α may be executed;
  • JAg is the set of Joint Actions such that every j ∈ JAg is a tuple hα1, α2, ..., αki where for each αi (i ≤ k) there is some i ∈ Ag such that αi ∈ Aci.
  • τ : Q x JAg → Q is a partial system transition function, which defines the state τ(q, j) that would result by the performance of j from state q - note that, as this function is partial, not all joint actions are possible in all states (cf. the precondition function above);
  • φ is a finite, non-empty set of atomic propositions; and
  • π : Q → 2φ is an interpretation function, which gives the set of primitive propositions satisfied in each state: if p ∈ π(q), then this means that the propositional variable p is satisfied (equivalently, true) in state q.
